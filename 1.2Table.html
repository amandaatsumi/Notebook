How to create aligned information

Use the tag Table

  • Use TR for Table Roll
  • Under every TR created a TD for Table Data
    • Every table data is equivalent to a additional column

How to add boards to the table

  1. Use tag TABLE
    • To add boards in HTML use table border="1"
  2. Use the tag THEAD for the header of the table
    • Use Tag TH to enter the header information
    • Each line TH is a for every column header
  3. Use the tag TBODY for the lines of the Table
    • Use TR for Table Rolls
    • Under every TR created a TD for line's data
  4. Use the tag TFOOT for the last line
    • This line can be use for totals, summaries, sums

The new complete table would look like this

TAGS DESCRIPTION
1.0 Tag Table To start a table
1.1 THEAD Refers to all the other table's columns
1.2 TH Insert information to the Header
2.1 TBODY Refers to all the other table's lines
2.2 TR Add a new line to the table
2.3 TD Insert information in the columns of the lines
