How to change items of the webpage

Tag Selectors

You are able to set a style for all the same elements in the website

  • Header 1: All of them be green ➟ h1{color: green;}
  • Images with a color background ➟ img{background-color: red;}
  • Images height and width ➟ img{height: 5px; width: 5px;}

    • Always respect the format Selector {property: value;}

Class Selectors

  • Applicable for changes in the same structure throughout the website
  • In other words, it is used for the same element repeatededly
    • HTML: To change only in one image ➟ "img class="name1"src="link" alt="name of the image
    • CSS: To specify a style ➟ .name1{color: red;}
  • An element can have more than one class
    • HTML: To change only in one image ➟ "img class="name1 name2"src="link" alt="name of the image
    • CSS: To specify a style ➟ .name1{color: red;} and .name2{border-radius: 100%}
    Pseudo Class

    They are the items on CSS References that has the : at the beginnning

    • :Hover ➟ enrance the image when the mouse moves over the image
    • CSS: element:hover {} ➟ a: hover {}

ID Selectors

  • Usually used for one specific element
  • HTML: to change one header 1 ➟ <"h1 id="name3">
  • CSS: to specify the style ➟ #name3{}
  • You can only use an ID once, and if it is duplicated, it will show as red in the line of the code
  • Can only have one ID for an element
