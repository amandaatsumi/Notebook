Changing Styles

Simple and local way

  • Add to the "BODY" and/or the part you want to change Style="background-color: #E2D5D5"

Changing the entire structute

This change need to be done in the HEAD area of the HTML code

  • Inside the Head tag, add tag Style
    • Inside Style, choose the part in the body you will change
    • Include Body{}
    • Examples of parts changed inside the {}
      • Background-color: blue;
      • One part changed inside the {}
    • Sometimes it is necessary to check the default CSS values and change them
      • Changing border-style to none
      • Height: 2px, for lines
      • Width: 30% or 200px. The percentage is convinient for cellphones, pc or differents devices
      • Border-style will always follow Top Right Bottom Left, so it could be
      • The https://devdocs.io can show you the details

Best way to change the styles

  • Create a New folder css and a file styles.css
  • add to this folder all the changes in background-color, text colour
  • Create in the Head a Link tag related to this new file
  • Copy the link reference to all pages and they'll all reflect the same style you created
