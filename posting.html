How to Post on Github

  1. Create a New Repository
  2. Don't forget to check Initialize this repository with README
  3. Hit Upload
  4. select all the files of the website
  5. Add a comment in the <Commit Changes
  6. Check Commit directly to the main branch
  7. Click Settings and in Github Pages
    • Change the source to Main and Docs
  8. The Link will be available to be shared
