How to add a list Like the Skills

  1. Use the tag thead for the headers of the table
    • Use Tag TH to enter the header information
    • Create one line of TH for every column header
  2. Use the tag tbody for the lines of the Table
  3. Under every TR created a TD for Table line
  4. Every table data TD is equivalent to a additional column
  5. Evey cell with characters must be a TD with the characters ★★★★☆
  6. Use TR for Table Roll
