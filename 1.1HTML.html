Basics HTML (Hyper Text Mark up Language)

General Notes

  1. To add line by line of the body - P
  2. To Add a line use HR and you can add more items as
    • Color = colour
    • Align = left by default
    • Noshade = remove shade
    • Size = height in pixels
  3. To look for details of how to use the tags, use https://devdocs.io
  4. Docs.emmet is the website that we use to get information from the shortcuts

How to create lists/Topics

  1. Select OL or UL for Ordered or unordered lists
  2. Enter every element in the list with li

How to add an images

  1. Save the image in the save directive you are saving your project with .png
  2. Use IMG tag and past the source, or enter the name of the file
    • Fill ALT with the description of the image, in case the image doesn't come up

Working with the Body Text

  1. Italisize= EM
  2. Bold = Strong
  3. Underscored = u

How to Create Hyperlink

  1. Use the Anchor Tag A
  2. Enter the addres in the ""
  3. Put the link "name" in the = <>
